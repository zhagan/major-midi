Transfer MIDI

Send `.mid` files directly to `0:/midi` on the Major MIDI SD card over USB MIDI SysEx.

Before You Start

  • Use a Chromium-based browser with Web MIDI SysEx enabled.
  • Connect the module over USB and power it normally.
  • Keep the SD card inserted while uploading.

Connect

Web MIDI permission not granted yet.

Upload

Drop `.mid` or `.midi` files here

Files are uploaded one at a time and written to 0:/midi.

Idle.

    Session Log

    What It Does

    This page uses a Major MIDI-specific SysEx upload protocol. Each file is sent in small acknowledged chunks so the module can write safely to the SD card without overrunning USB MIDI receive.

    Behavior Result
    Upload target 0:/midi/<filename>
    Existing file name match Overwritten
    Accepted files .mid and .midi
    After a completed upload Major MIDI rescans the media library automatically

    Requirements

    Web MIDI SysEx is not available in every browser. If the page cannot request SysEx permission, try current Chrome or Edge on a secure origin.

    This page talks to the USB MIDI interface on the module. It does not mount the SD card directly.
