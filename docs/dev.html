Dev Resources

Build notes, source map, and development entry points.

Overview

Developer-facing GitHub documentation still lives in README.md. This page is the site-facing entry point for people who need build context, source layout, and docs workflow.

Build

Major MIDI uses the libDaisy Make workflow.

make

Useful follow-up:

make clean
python3 docs/generate_docs.py

Source Layout

PathPurpose
src/main.cppApp setup, transport loop, media reload, save flow
src/ui/Input scanning, event translation, UI state changes, OLED rendering
src/midi/MIDI file playback, routing, media browser, scheduler
src/synth/SF2 synth integration
src/cv/CV/gate behavior
src/persist/Boot state and song-scoped persistence
docs/Generated site output and CSS
